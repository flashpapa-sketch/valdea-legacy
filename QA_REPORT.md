## 🔧 修正内容

### ■修正1：◆三 武器屋の論理的矛盾を修正

**問題点:**
- テキスト：「三種類の武器がある。短剣（金貨三枚）、剣（金貨八枚）、手斧（金貨五枚）」
- 実装：短剣の選択肢のみ存在

**修正内容:**
```
変更後: もし余裕があれば金貨三枚でこの短剣を買ってもよい。
```

**理由:**
- より自然で柔らかい表現
- 「店内は棚の半分が空だ」という記述にも整合
- Sorcery!スタイルの二人称・現在形に合致

---

### ■修正2：ステータス決定スキップ問題を修正

**問題点:**
- 新規ゲーム開始時、サイコロを振らずに進めてしまう
- デフォルト値（SK=7, HP=14, LK=7）で自動的にゲームが始まる

**修正内容:**
```
index.html:
- 「◆ 冒険を始める」→ guide.html（新規ゲーム、ステータス決定必須）
- 「◆ 冒険を再開する」→ game.html（セーブデータから再開）
  ※セーブデータがある時のみ表示
```

**理由:**
- ステータス決定を必須化
- 新規プレイヤーは必ずguide.htmlでサイコロを振る
- 継続プレイヤーはワンクリックで再開可能

## 📝 検出されたバグと修正履歴

### バグ1：未定義フラグ10個（誤検出）
- **検出:** 初版QA
- **原因:** QAスクリプトがsetFlag()を探していたが実際はflag()
- **結論:** 実装は正常、バグではない

### バグ2：到達不能パラグラフ24個（誤検出）
- **検出:** 初版QA
- **原因:** QAスクリプトがpayGoto()、luckCheck()を追跡していなかった
- **結論:** 全108パラグラフが到達可能、バグではない

### バグ3：◆三 武器屋の論理的矛盾（実バグ）
- **検出:** 人間による確認
- **内容:** テキストと選択肢の不一致
- **修正:** テキストを「もし余裕があれば金貨三枚でこの短剣を買ってもよい」に修正 ✅

### バグ4：ステータス決定スキップ問題（実バグ）
- **検出:** 人間による確認
- **内容:** サイコロを振らずにゲームを開始できてしまう
- **修正:** index.htmlを「冒険を始める」「冒険を再開する」に分離、guide.html必須化 ✅
