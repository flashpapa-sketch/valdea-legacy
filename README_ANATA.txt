 Sorcery!スタイルの特徴
==========================================

✓ 簡潔で直接的な文体
✓ 環境描写を冒頭に配置
✓ 五感に訴える表現
✓ 「――」（ダッシュ）で雰囲気を演出
✓ 「あなたは〜もできれば〜こともできる」

例：
【変更前】
あなたはその場に倒れた。

【Sorcery!スタイル】
あなたは倒れた。
冷たい石の床に頬をつけたまま、意識が遠ざかっていく。
