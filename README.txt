【プレビュー方法】

■ 方法1：Python（推奨）
  1. このフォルダでターミナルを開く
  2. python3 start_server.py を実行
  3. ブラウザが自動で開く
     → http://localhost:8080/game.html?new=1

■ 方法2：VS Code
  1. VS Codeで game.html を開く
  2. 右クリック → "Open with Live Server"

■ GitHub Pages
  index.html / game.html / guide.html を
  リポジトリのルートにアップロードするだけ
