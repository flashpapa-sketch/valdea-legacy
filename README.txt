リリース版 v1.0

■ 同梱ファイル
- index.html  : タイトルページ（日英切替対応）
- game.html   : ゲーム本体（108パラグラフ／115戦闘／7エンディング）
- guide.html  : 冒険者の手引き

■ オフラインでの遊び方
1. ZIPを解凍
2. index.html をダブルクリック → ブラウザで開く
3. 「◆ 冒険を始める ◆」をクリック
※ インターネット不要。localStorageにセーブが保存されるため、
  同じブラウザなら続きから再開可能。
