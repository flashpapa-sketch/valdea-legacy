 実装済みシステム詳細
==========================================

【冒険者データ】
- 技術点（SK）：戦闘での命中率に影響
- 体力点（HP）：0以下でゲームオーバー
- 運点（LK）：運試しの成功率に影響
- 経験値（XP）：レベルアップに必要
- レベル（LV）：パラメータボーナス

【戦闘システム】
- 敵27種類
- 戦闘箇所28ポイント
- 敵統計値：SK=1-20, HP=1-50, XP=0-20
- HPマイナス防止：実装済み
- 連打防止：実装済み

【特殊システム】
- 猿の右手：一度限りの切り札（12種結果）
- 腐敗値（corrupt）：0-3段階
- 関係値（rel）：ナック/ゴルム/シラ
- フラグ管理：ロサ死亡/ラルハス/ヴァズ他

【ラルハス伏線】
1. ◆三十六～三十七：初出会い→背後霊
2. ◆二百十五：戦闘で接触
3. ◆ラルハス裏切り：遺跡で裏切り（corrupt+1）
4. ◆ラルハス再登場：最深部で再会
5. ◆四百十五：ラスボス前の最後の選択

【エンディング分岐】

- ◆四百三十七の二：TRUE END（星脈継承）
