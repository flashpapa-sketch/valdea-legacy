# ヴァルデアの遺産 節追加スキル

## 概要
scenes_full.jsonに新しい節を追加し、game.htmlに反映する。

## 事前確認
- 追加する節番号（例：五百一）
- 節の本文テキスト
- 選択肢とジャンプ先節番号
- ゲームメモ（体力変化・アイテム取得など）があればnoteに記載

## JSONの節データ形式
```json
{
  "節番号": {
    "text": ["本文段落1", "本文段落2"],
    "note": "＊ゲームメモ（省略可）",
    "choices": [
      {"label": "選択肢テキスト→節番号へ進め", "next": "節番号"}
    ]
  }
}
```

## エンディング節の形式
```json
{
  "節番号": {
    "text": ["本文"],
    "end": "GOOD END 1「タイトル」"
  }
}
```

## 手順
1. /home/claude/scenes_full.json を読み込む
2. 新しい節データをJSONに追加
3. game.htmlを scenes_full.json から再生成
4. 全チェックスキルを実行
5. 問題なければZIP作成をユーザーに確認

## 体力変化ボタンの自動検出パターン
節のテキストに以下が含まれる場合、自動ボタンを生成する：
- 「体力点を〇点回復」→ hp_plus
- 「体力点を〇点失」→ hp_minus
- 「技術点に〇点を加」→ sk_plus
- 「技術点を〇点減」→ sk_minus
- 「運勢点に〇点を加」→ lk_plus
- 「金貨〇枚を失」→ gold_minus

## 注意事項
- 節番号は漢数字（一・二・三...百・二百...）
- ジャンプ先節番号は必ず既存の節か今回追加する節であること
- 追加後は必ず全チェックスキルでリンク切れを確認
